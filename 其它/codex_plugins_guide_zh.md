## 通用使用方法

大多数插件的用法不是“手动点按钮”，而是在 Codex/ChatGPT 里用自然语言提出目标：

```text
帮我读取这个项目的 GitHub PR，整理未解决评论并提出修改方案。
把这份 CSV 做成分析表格，并生成三页汇报 PPT。
读取我的 Google Calendar，整理下周会议冲突和准备事项。
在 Slack 里查找某个项目最近一周的讨论，总结成行动清单。
检查 Sentry 最近的错误，找出最可能的代码原因。
```

使用插件时建议说明四件事：

1. 数据范围：哪个项目、哪个仓库、哪个文件夹、哪个时间段。
2. 期望产物：总结、表格、邮件草稿、PR、图表、任务列表等。
3. 输出格式：Markdown、表格、PPT、DOCX、CSV、Issue 评论等。
4. 权限边界：只读、先确认再修改、允许创建文件或发消息。

## Featured

### Spreadsheets

- 截图说明翻译：创建和编辑电子表格文件。
- 功能：处理 `.xlsx`、`.csv`、表格清洗、公式、图表、透视分析、报表生成。
- 典型用法：让 Codex 读取销售数据，生成汇总表、趋势图和异常值说明。
- 适合：财务、运营、数据分析、项目汇报。
- 热门与实用性：S。通用办公高频刚需，几乎所有团队都会用到。

### Presentations

- 截图说明翻译：创建和编辑演示文稿。
- 功能：生成 PPT、改写页面文案、统一版式、插入图表、做汇报结构。
- 典型用法：把一份分析报告压缩成 10 页管理层汇报。
- 适合：汇报、路演、教学、产品方案。
- 热门与实用性：S。对“把资料变成可讲的内容”特别有价值。

## Coding

- 截图说明翻译：检查模型、数据集、Spaces 和研究内容。

- 功能：BI 查询、指标口径、数据探索、图表和报表。
- 典型用法：基于统一指标口径生成本周营收分析。
- 适合：数据团队、业务分析。
- 热门与实用性：B。BI 场景很有价值，但取决于团队是否使用 Omni。

### Pylon

- 截图说明翻译：访问 Pylon 客户支持平台。
- 功能：客户支持工单、B2B 客户问题、Slack/邮件支持流。
- 典型用法：总结某企业客户过去 30 天的所有支持请求。
- 适合：B2B SaaS 客服和客户成功。
- 热门与实用性：B。B2B 支持场景垂直。

### Pipedrive

- 截图说明翻译：同步 Pipedrive 交易和联系人。
- 功能：CRM、销售 pipeline、联系人、交易阶段、跟进任务。
- 典型用法：找出停滞超过 14 天的 deals，并起草跟进邮件。
- 适合：销售团队、中小企业 CRM。
- 热门与实用性：A。销售 CRM 中知名且实用。

## 相似插件怎么选

### 邮件和日历

- Gmail / Google Calendar：适合 Google Workspace 用户。
- Outlook Email / Outlook Calendar：适合 Microsoft 365 用户。
- Fyxer：更偏邮件写作和收件箱助理，和邮箱插件有重叠。

### 团队沟通和会议

- Slack：科技和创业团队常见。
- Teams：企业和微软生态常见。
- Fireflies / Otter.ai / Granola / Circleback：都偏会议记录、转写、总结和行动项。已经有其中一个时，通常不需要同时启用太多。

### 项目管理

- Linear：工程团队、产品研发节奏强。
- ClickUp / Monday.com：更综合，适合跨部门项目管理。
- Atlassian Rovo：Jira + Confluence 企业环境优先。

### 文档和云盘

- Google Drive：Google 生态首选。
- SharePoint：微软企业生态首选。
- Box / Egnyte：企业文件管理和合规场景。
- Documents / Spreadsheets / Presentations：更偏直接创建和编辑本地或工作区产物。

### 部署平台

- Vercel：Next.js 和前端全栈优先。
- Netlify：静态站、Jamstack、前端项目。
- Render：后端服务、数据库、小型全栈应用。
- Cloudflare：边缘、DNS、CDN、Workers、Pages。
- Hostinger：更偏网站托管和小企业建站。

### CRM 和销售

- HubSpot：综合 CRM 和营销自动化，最通用。
- Pipedrive：销售 pipeline 轻量好用。
- Attio：现代 CRM 和关系型数据体验。
- HighLevel：营销代理商和本地业务。
- Carta CRM：投资行业。
- Pylon / Help Scout：客户支持，不完全等同 CRM。

### 数据和分析

- Spreadsheets：最通用，适合文件级分析。
- MotherDuck：SQL 和 DuckDB 数据分析。
- Omni Analytics：BI 和语义层。
- Amplitude：产品行为分析。
- Coupler.io：多数据源同步和报表。
- Vantage：云成本分析。

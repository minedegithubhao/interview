**项目背景**：新闻内容投递依赖人工分类时，存在成本高、效率低、分类稳定性不足的问题。项目目标是根据新闻标题自动识别频道类别，实现新闻自动分类投递，从而提升内容分发效率和用户粘性。

**技术栈**：Flask、随机森林、FastText、BERT、Transformer、模型量化、知识蒸馏。

**个人职责**：

- 建设新闻标题分类能力，将新闻文本自动映射到对应频道类别，为内容投放、推荐和运营分发提供基础标签。
- 对比机器学习与深度学习方案，使用随机森林、FastText 以及基于 Transformer 架构的 BERT 进行分类建模与效果优化。
- 通过 Flask API 对外提供新闻分类能力，便于业务系统调用和后续集成。
- 围绕分类准确率、人工成本和投递效率等指标持续优化模型方案和服务调用链路。

**项目成果**：

- 分类准确率达到 **92%**，相较原方案准确率提升约 **20%**。
- 将人工分类成本降至原来的 **20%**，实现新闻分类流程的降本增效。
- 支撑新闻自动分类投递，提高内容分发效率和用户粘性。

# 深度学习

核心4问：

1. **能不能优化下来**：梯度、反向传播、学习率、优化器、初始化。
2. **能不能泛化出去**：过拟合、正则化、数据增强、验证集。
3. **训练是否稳定**：梯度消失/爆炸、归一化、梯度裁剪、混合精度。
4. **训练是否高效**：batch size、显存、分布式、梯度累积、checkpoint。

### 深度学习方法论

`2 2 2 5 原则`

2 

1. 构建数据集对象
2. 构建数据加载器

2

1. 构建损失函数
2. 构建优化器

2

1. 遍历训练轮次
2. 遍历数据加载器

5

1. 前向传播
2. 计算损失
3. 梯度清零
4. 反向传播
5. 参数更新

## **反向传播是什么？**

反向传播是**基于链式法则，从损失函数出发，沿计算图反向高效计算各层参数梯度的过程。**

拔高：

> 得到梯度后，优化器再根据更新规则，比如 SGD 或 Adam，对参数进行更新。

## BGD、SGD和Mini-Batch的区别是什么？

- BGD 批量梯度下降（全量数据）：用**全部数据集**算梯度更新
  - 优点：梯度方向稳定
  - 缺点：参数更新慢、显存压力大
  - 适用：小数据集

- SGD 随机梯度下降（单样本）：每次只用**1 个样本**更新
  - 优点：参数更新快
  - 缺点：梯度噪音大、震荡严重、难精准收敛
- Mini-Batch 小批量梯度下降（**工业首选**）：每次取**一小批样本**更新
  - 优点：兼顾速度 + 梯度稳定、收敛快又平稳、显存友好

## 深度学习模型训练步骤

```python
# 深度学习模型训练步骤
# 1. 获取数据
# 2. 构建模型
# 3. 构建优化器  optimizer.优化器(模型的参数, 学习率)
# 4. 计算损失 实例化损失函数  loss_f(预测值, 真实值)
# 5. 梯度清零 原因：上一轮的梯度不能用  optimizer.zero_grad()
# 6. 反向传播（计算梯度） loss.backward()
# 7. 参数更新 optimizer.step()

class Model(nn.Module):
    def __init__(self):
        super().__init__()
        self.linear1 = nn.Linear(3, 6)
        self.linear2 = nn.Linear(6, 6)
        self.outer = nn.Linear(6, 1)

    def forward(self, x):
        x = torch.sigmoid(self.linear1(x))
        x = torch.sigmoid(self.linear2(x))
        x = torch.sigmoid(self.outer(x))
        return x


if __name__ == '__main__':
    # 准备训练数据
    x = torch.randn(4, 3)
    # 实例化模型
    model = Model()
    print('start：', model.state_dict())
    # 训练模型
    y_pred = model(x)
    # 预测
    y_true = torch.tensor([[0.], [1], [0], [1]])

    # 构建梯度优化器
    optimizer = torch.optim.SGD(model.parameters(), lr=0.1)

    # 计算损失
    criterion = nn.BCELoss()
    loss = criterion(y_pred, y_true)

    # 梯度清零
    optimizer.zero_grad()

    # 反向传播
    loss.backward()

    # 参数更新
    optimizer.step()

    print('end：', model.state_dict())
```
